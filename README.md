# ♻️ 분리수거 분류 챗봇

## 📌 프로젝트 소개
이미지를 입력받아 쓰레기 종류를 분류하고,  
올바른 분리수거 방법을 안내하는 딥러닝 기반 챗봇입니다.

---

## 🚀 주요 기능
- 📷 이미지 업로드를 통한 쓰레기 분류
- 🤖 챗봇을 통한 분리수거 방법 안내
- 🇰🇷 한국 분리수거 기준 정보 제공

---

## 🛠 기술 스택
- Python 3.11
- TensorFlow / Keras
- Streamlit
- OpenAI API

---

## 📊 모델 설명
- 모델: ResNet50
- Optimizer: Adam
- Loss: Categorical Crossentropy
- 입력 이미지 크기: 224x224
